# Colors

White, Black, and Progressive Red: these are the Audi brand colors. The order is immaterial because
there is no hierarchy here. What matters is that the colors support the message you want to convey and that
the overall layout is not too colorful.

## Audi color palette

### Audi brand colors

The Audi brand colors are equal in status and always appear in full tone. They may not be darkened, lightened, or displayed transparently.

| Color     | Name            | Value   |
| --------- | --------------- | ------- |
| Header    | White           | #FFFFFF |
| Paragraph | Black           | #000000 |
| Paragraph | Progressive Red | #F50537 |

### Functional Grey Tones

For functional applications, the color palette can be extended to include grey tones ranging from white to black.

| Color     | Name    | Value   |
| --------- | ------- | ------- |
| Header    | Grey 5  | #F2F2F2 |
| Paragraph | Grey 10 | #E5E5E5 |
| Paragraph | Grey 15 | #D9D9D9 |

## Audi signal colors

The Audi signal colors are used solely for notifications, warnings, and status messages if a color differentiation is
required. Red can be used in blocks, while yellow and green may only be used for typography and icons.

| Color     | Name          | Value   |
| --------- | ------------- | ------- |
| Header    | Signal Red    | #BB0A30 |
| Paragraph | Signal Yellow | #009900 |
| Paragraph | Signal Green  | #FFAA00 |

## Accessibility

When using colors or gradients, contrast must pass WCAG AA or higher. To confirm the accessibility of the colors or
gradients being utilized, users can utilize a Contrast Checker.

It is important to note that the size of the text is a factor when scaling accessibility for contrast. For example,
light text on a dark background may pass AA guidelines at 12px, but users may need a stronger font weight or size for a 10px
footnote using the same color combinations.
